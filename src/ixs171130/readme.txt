Compiling:

1. After unzipping, go to the folder containing the project folder (ixs171130)
2. Compile using `javac ixs171130/Num.java`
3. To run with demo driver, use `java ixs171130.Num`

Running:
If using included driver, give space between every operator and digit. `( 5 + 5 )` will work. `(5 + 2)` will not.
